Row

The Row splits Column components into vertical oriented boxes. A Row should only contain Column components to avoid appearance issues. A Row contains one or more Column components and will break to a new line if one or more Column components exceed in size. Below you will see a code example to get an idea of how it should look like.

<Container>
  <Row>
    <Column />
  </Row>
</Container>
