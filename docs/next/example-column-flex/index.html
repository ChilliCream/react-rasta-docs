Column Flexibility

In this example column's sizes are set to auto or none and column's flex is set to grow, shrink or none. If you would like to see how it behaves during resizing, click here.

Code

<Container>
  <Row>
    <Column size={"none"} flex={"grow"}>
      grow grow grow grow
    </Column>
    <Column size={"auto"} flex={"shrink"}>
      shrink
    </Column>
    <Column size={"none"} flex={"none"}>
      none
    </Column>
    <Column size={"none"}>undefined</Column>
  </Row>
</Container>
