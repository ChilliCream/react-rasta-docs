Break

A Break forces Columns inside a Row to split into new rows even if the Columns do not exceed in size. A Break should not contain any children and has no properties at all. Below you will see how it works.

<Container>
  <Row>
    <Column size="3">Top</Column>
    <Break />
    <Column size="3">Bottom</Column>
  </Row>
</Container>
